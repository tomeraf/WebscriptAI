Quick Start Guide

Basic Usage

  1. Click the extension icon to open the popup
  2. Toggle the "Enable Script Runner" switch
  3. Write your JavaScript in the "Initial Script" textarea
  4. Set a delay (optional) and click away to auto-save
  5. Reload the page to run your script

Using the AI Assistant

  1. Click "Chat with AI Assistant" in the popup
  2. Enter your API key when prompted (one-time setup)
  3. Describe what you want: "Find all images and make them grayscale"
  4. Click "Use as Initial Script" or "Use as Repeat Script"
  5. The script is automatically saved to your extension

Repeating Scripts

Use the "Repeat Script" section to run code at regular intervals:

  • Enter your script in the "Repeat Script" textarea
  • Set the interval (in seconds)
  • Perfect for monitoring, polling, or continuous automation

Example Scripts

Remove all ads

document.querySelectorAll('[id*="ad"], [class*="ad"]').forEach(el => el.remove());

Auto-scroll to bottom

window.scrollTo(0, document.body.scrollHeight);

Extract all links

Array.from(document.querySelectorAll('a')).map(a => a.href);

Dark mode any website

document.body.style.filter = 'invert(1) hue-rotate(180deg)';

FAQ

Is this free?

Yes! The extension is free and open source. The Gemini API has a generous free tier.

Is my data safe?

Absolutely. All scripts run locally in your browser. Your API keys are stored locally. We use Vercel only for error logging (anonymous).

Can I use this for web scraping?

Yes, but respect website terms of service and rate limits. Use responsibly.

Why isn't my script working?

Make sure the extension is enabled, the page is reloaded, and check the browser console for errors.

Can I contribute?

Yes! This is open source. Submit issues, pull requests, or suggestions on GitHub.
